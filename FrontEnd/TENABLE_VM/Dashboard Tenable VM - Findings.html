Guia de Exportação - Tenable One

Para garantir que o dashboard de Findings apresente todas as informações e gráficos corretamente, siga os passos abaixo ao exportar seus dados:

  1. Acesse a plataforma Tenable One.
  2. Navegue até o menu de Vulnerabilidades (ex: Explore > Findings).
  3. Filtre o State (Status) para trazer pelo menos ACTIVE, NEW e RESURFACED (e FIXED, se quiser ver a evolução mensal completa).
  4. Clique no botão de Exportar e selecione o formato CSV ou Excel.
  5. Na janela que se abrir, garanta que as colunas abaixo estejam marcadas:
Severity
State
VPR Score
EPSS Score
Asset Operating System
Plugin Name
Solution
Plugin Family
Patch Published
First Observed
Last Seen

6. Clique em exportar, aguarde o download e faça o upload do arquivo gerado aqui no painel.
