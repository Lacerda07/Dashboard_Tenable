Guia de Exportação - Tenable VM

Para garantir que o dashboard de Agentes apresente todas as informações e gráficos corretamente, siga os passos abaixo ao exportar seus dados:

  1. Acesse a plataforma do Tenable VM.
  2. No menu lateral, navegue até a aba correspondente aos seus Agentes (ex: Sensors > Agents).
  3. Aplique os filtros desejados (ex: Grupos específicos ou todos os agentes).
  4. Clique no botão de Exportar (ícone de download na direita) e selecione o formato CSV.
  5. Na janela que se abrir, marque obrigatoriamente as seguintes colunas:
Status
Platform
Health
Agent Name
Linked On
Last Scanned
IP Address
Version
Groups
Last Connect

6. Clique em exportar, aguarde o download e faça o upload do arquivo gerado aqui no painel.
