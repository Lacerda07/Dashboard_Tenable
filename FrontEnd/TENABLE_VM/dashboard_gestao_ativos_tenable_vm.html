Guia de Exportação - Tenable VM

Para garantir que o dashboard apresente todas as informações e gráficos corretamente, siga os passos abaixo ao exportar seus ativos:

  1. Acesse a plataforma do Tenable VM.
  2. No menu lateral, navegue até a aba Explore > Assets.
  3. Aplique os filtros desejados (ex: Tags específicas ou todos os ativos).
  4. Clique no botão de Exportar (ícone de download na direita) e selecione o formato CSV.
  5. Na janela que se abrir, marque obrigatoriamente as seguintes colunas:
    • id
    • name
    • is_licensed
    • first_observed
    • last_observed
    • last_authenticated_scan_time
    • sources
    • types
    • acr.score
    • aes.score
    • display_operating_system
    • is_public
  6. Clique em exportar, aguarde o download e faça o upload do arquivo gerado aqui no painel.
